# ComboPaste

Free temporary pastebin for combolist sharing.

🌐 **Website:** [https://combopaste.site](https://combopaste.site)

Paste your list, set auto-expiry, and get a shareable link. No account needed, no logs.

## Features

- Temporary hosting with auto-expiry (1h / 6h / 1d / 7d / 30d / Forever)
- No account or registration required
- No logs or tracking
- Cloudflare protected
- Anonymous

## How it works

1. Go to [combopaste.site](https://combopaste.site)
2. Paste your combolist or upload a .txt file
3. Choose auto-delete time
4. Get a shareable link

## Tech

- Cloudflare Workers (API + proxy, origin hidden)
- GitHub Pages (paste storage)
